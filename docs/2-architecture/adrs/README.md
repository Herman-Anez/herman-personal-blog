# Architecture Decision Records

This directory contains the project's ADRs, managed with [Log4brains](https://github.com/thomvaill/log4brains).

To navigate ADRs, consult the [ADR index](./index.md).

## Development

If not installed, install Log4brains:

```bash
npm install -g log4brains
```

To preview the knowledge base locally:

```bash
log4brains preview
```

To create a new ADR interactively:

```bash
log4brains adr new
```
