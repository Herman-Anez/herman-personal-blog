# <T es="Sistema de Traducción" en="Translation System" />

Este post sirve como referencia para el uso del componente `<T />`.

## 1. Traducción por ID (JSON Modular)
Este texto viene del archivo `ui.json`:
**<T id="ui.techStack" />**

Este viene de `blog.json`:
**<T id="blog.test.title" />** -> *<T id="blog.test.description" />*

### 1.1 Inyección Directa (¡Nuevo y más fácil!)
Ahora puedes usar el diccionario directamente como una variable `d`:
- Desde `ui.json`: **`{d.ui.techStack}`**
- Desde `blog.json`: **`{d.blog.test.title}`**

### 1.2 Superpoder de Fallback
Si usas un ID que no existe, pero defines texto inline, el componente `<T />` te salva:
**<T id="llave.fantasma" es="Fallback Inline" />**
*(Aquí intentó buscar 'llave.fantasma', falló y usó el texto en español proporcionado).*
